Proyecto de tienda en línea con funcionalidades completas.

Trabajo integrador utilizando Pandas para análisis de datos.

Color Game

To Do List

Lista de Tareas por Hacer

Este proyecto es una aplicación web simple que es ideal para oraganizar tus tareas pendientes de manera interactiva. Creada con HTML, CSS y JavaScript, esta aplicación te permite Agregar Tareas, Editar Tareas y Eliminar Tareas de manera facil e intuitiva. Además posee un Contador de Tareas Pendientes.

Sistema de Gestión de Turnos

Gestión de una Estética

El Sistema Gestor de Turnos es una aplicación web para gestionar usuarios, empleados y turnos, permitiendo agregar, actualizar y eliminar registros. Está desarrollado con Flask para el backend, MySQL como base de datos, y Bootstrap para la interfaz responsiva. Utiliza Python para la lógica del servidor y JavaScript para interactividad en el frontend.
